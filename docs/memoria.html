Introducción

Contexto

DrumStore Pro es una tienda online ficticia especializada en productos de percusión. El proyecto se desarrolla como ejercicio práctico del módulo de Diseño de Interfaces Web.

Objetivos

  • Diseñar una interfaz clara y accesible.
  • Aplicar buenas prácticas de maquetación.
  • Optimizar la experiencia en dispositivos móviles.
  • Trabajar SEO básico y estructura semántica.

Planificación

El proyecto se ha dividido en varias fases: análisis, diseño, implementación, validación y documentación.

Fase Contenido
Análisis Definición de objetivos y público
Diseño Wireframes y guía visual
Implementación HTML, CSS y Bootstrap
Validación Accesibilidad y pruebas

Desarrollo

Estructura del proyecto

Herramientas

  • HTML5 semántico
  • Bootstrap 5.3

  • JavaScript básico

Accesibilidad

Se ha trabajado el contraste, el uso correcto de etiquetas, la navegación por teclado y los textos alternativos.

  • Foco visible
  • Alt en imágenes
  • Estructura de encabezados
  • Formularios etiquetados

Usabilidad

La interfaz sigue patrones habituales de comercio electrónico, facilitando la navegación y el proceso de búsqueda.

  • Menú claro
  • Categorías visibles
  • Botones reconocibles
  • Feedback visual

SEO

Se han aplicado técnicas básicas de optimización para buscadores.

  • Meta etiquetas
  • Títulos jerárquicos
  • URLs limpias
  • Schema básico

Conclusiones

El proyecto DrumStore Pro permite aplicar de forma práctica los conocimientos adquiridos en el módulo de DIW, desarrollando una interfaz moderna, accesible y preparada para entornos reales.
